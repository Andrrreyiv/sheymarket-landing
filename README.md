# Швеймаркет — Landing Page

Лендинг для компании Швеймаркет — оптовый пошив одежды и поставщик фурнитуры (Нальчик, КБР).

## Структура

```
sheymarket-landing/
├── index.html          # главная и единственная страница
├── css/
│   └── styles.css      # все стили (CSS variables + flat structure)
├── js/
│   └── main.js         # бургер, формы, scroll, анимация при скролле
└── assets/
    └── img/
        ├── hero.jpg    # фото для первого экрана
        └── showroom.jpg# фото для блока «о нас»
```

## Запуск

Открыть `index.html` в браузере либо разместить файлы на любом хостинге (например, GitHub Pages, Netlify, Cloudflare Pages, VPS).

Никаких сборщиков и зависимостей — только HTML/CSS/JS.

## Адаптив

- **Desktop:** ≥ 1100px
- **Tablet:** 600–1100px
- **Mobile:** < 600px

## Структура страницы (14 блоков)

1. Topbar + Header (sticky)
2. Hero — H1 + фото + мини-форма захвата
3. About — 3 вопроса «для кого мы» (жёлтые/белые карточки)
4. Clients — «Почему Лента и Светофор 5 лет с нами»
5. Testimonial — отзыв Алокова Гисы
6. Catalog — топ-12 видов одежды (SVG-иконки)
7. Plan — 4 шага сотрудничества с пунктирными стрелками
8. Furniture intro — переход к фурнитуре
9. Problems — 3 проблемы производителей + жёлтая плашка-решение
10. Assortment — облако чипов с категориями
11. Own production — топ-3 (тесьма, резинка, термоэтикетка)
12. Steps — 5 шагов навстречу клиенту
13. Metrics — 30% / 80% / 10 из 10 (на синем фоне)
14. Final CTA — «Вы ведь этого хотели?» + форма + мессенджеры
15. Footer
