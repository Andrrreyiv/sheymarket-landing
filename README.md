# Швеймаркет — Landing Page

Лендинг для компании Швеймаркет — оптовый пошив одежды и поставщик фурнитуры (Нальчик, КБР).

## Структура

```
sheymarket-landing/
├── index.html          # главная и единственная страница
├── css/
│   └── styles.css      # все стили (CSS variables + flat structure)
├── js/
│   └── main.js         # бургер, формы, scroll, анимация при скролле
└── assets/
    └── img/
        ├── hero.jpg    # фото для первого экрана
        └── showroom.jpg# фото для блока «о нас»
```

## Запуск

Открыть `index.html` в браузере либо разместить файлы на любом хостинге (например, GitHub Pages, Netlify, Cloudflare Pages, VPS).

Никаких сборщиков и зависимостей — только HTML/CSS/JS.

## Адаптив

- **Desktop:** ≥ 1100px
- **Tablet:** 600–1100px

## Структура страницы (14 блоков)

1. Topbar + Header (sticky) — wordmark «ШВЕЙМАРКЕТ» + иконка иглы
2. Hero — H1 + адрес КБР + 2 CTA + floating-форма поверх фото
3. About — 3 вопроса «для кого мы» (yarko-style: жёлтые карточки с чёрной рамкой 2.5px и тенью-смещением)
4. Clients — «Почему Лента и Светофор 5 лет с нами» (логотипы-приближения к оригиналам)
5. Testimonial — отзыв Алокова Гисы
6. Catalog — топ-12 видов одежды (SVG-иконки с цветовыми категориями)
7. Plan — 4 шага сотрудничества с крупными жёлтыми номерами + dashed-трасса
8. Furniture intro — переход к фурнитуре
9. Problems — 3 проблемы производителей + жёлтая плашка-решение (typography ladder: 60% + 10 лет)
10. Assortment — облако чипов с категориями
11. Own production — топ-3 (тесьма, резинка, термоэтикетка)
12. Steps — 5 шагов навстречу клиенту
13. Metrics — 30% / 80% / 10/10 крупными outline-цифрами на синем фоне
14. Bridge «Остались вопросы?» + Final CTA «Вы ведь этого хотели?» + форма + мессенджеры
15. Footer

## Доработки

Лендинг переработан по критике от заказчика — основные изменения:

- **Логотип:** чистый wordmark без подстрочника, иконка иглы-с-ниткой
- **Hero:** floating-карточка-форма поверх фото (как у risolut.ru), адрес КБР как осознанный элемент
- **Блок «Кому подходит»:** yarko-style — жёлтые карточки с черной 2.5px рамкой + UPPERCASE заголовки + жёлтый круг с галочкой
- **Логотипы клиентов:** переработаны под образ Ленты/Светофора
- **Метрики:** 96–120px outline-вариант на фоне крупных полупрозрачных цифр
- **Жёлтая плашка-решение:** typography ladder с акцентом на «60%» и «10 лет»
- **Шрифт:** Manrope вместо Inter
- **План работы:** крупные жёлтые номера 01–04 с outline + dashed-трасса между шагами
- **Scroll-reveal:** fade-up анимации со stagger 70ms
- **Floating-кнопка:** только на mobile
- **Pre-footer:** мост «Остались вопросы?»
- **OG-теги + SVG favicon**
